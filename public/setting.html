Hướng dẫn lấy Telegram Bot Token / Chat ID

  1. Vào Telegram, tìm @BotFather.
  2. Tạo bot mới bằng lệnh /newbot.
  3. Copy Bot Token đưa vào ô Telegram Bot Token.
  4. Nhắn một tin bất kỳ vào bot vừa tạo.
  5. Mở trình duyệt với link:
https://api.telegram.org/botBOT_TOKEN/getUpdates

Tìm trường:

chat.id

Đó là Chat ID cần nhập.
